Helpers

Helpers is independent functions that you can use in any part of your code.

Helpers stored in DIR_SYSTEM helper directory (/system/helper by default)

You can load helper file in model or controller with load object, use helper method
or you can include it yourself in /system/startup.php file.

Default helpers

function token(int $length = 32) : string

Create random token with defined length. Not cryptographically safe, it uses mt_rand function.

UTF-8 helpers

A bunch of functions to process string with utf-8 text.

Basically, this function are wrappers for mbstring functions, but if mbstring extension is not loaded, it will try use iconv extension functions instead.
They are used in opencart default files often.

function utf8_strlen(string $string) : int
If iconv: can return false if there's encoding error
function utf8_strpos(string $string, int $needle, int $offset) : false|int
function utf8_strrpos(string $string, int $needle, int $offset) : false|int
function utf8_substr(string $string, int $offset, int $length) : string
If iconv: returns false if $offset is greater than $string length
function utf8_strtoupper(string $string) : string
If iconv: returns false if string is empty
function utf8_strtolower(string $string) : string
If iconv: returns false if string is empty
