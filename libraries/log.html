Log Library

Log library helps you write logs.

Log is not loaded with load object library.
You create Log instance yourself.
new Log('my-log.log')

Properties

private resource|false $handle
Log file handle.

Methods

public function __construct(string $filename)
Opens a file handle by $filename in DIR_LOGS (/system/logs by default)
public function write(string $message)
Write message to the file on new line including time
public function __destruct()
Closes file handle
