Encryption Library

With encryption library, you can encrypt and decrypt data.

Encryption is registry-stored library, call it by $this->encryption

Methods

public function encrypt(string $value) : string
Encrypt data
public function decrypt(string $value) : string
Decrypt data

Properties

private $key
Encryption passphrase. Configures in admin, setting config_encryption. Warning! If you change your passphrase, you will lost all encrypted data. So decrypt data and save it before changing passphrase.
