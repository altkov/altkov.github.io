Response Library

Response library used to take handle of response to user

Response is registry-stored library. In controller (and you must use it only in controller) you can call it by $this->response

Methods

public function addHeader(string $header)
Adds header to queue (it's not added immediately!). Method output enqueues all the headers. It's called in opencart core, in DIR_SYSTEM framework.php (system/framework.php by default).
public function redirect(string $url, int $status = 302)
Redirect user to $url with passed $status code and stop script execution.
Unlike of addHeader, redirects immediately.
public function setOutput(string $output)
Sets given string as response output. If called again, output changes.
public function getOutput() : string|null
Returns output, that was set in setOutput method
public function output()
Sends all the output to browser. Called in core file DIR_SYSTEM framework.php (/system/framework.php), so you don't have to call it.
public function setCompression(int $level = 0)
Function used in core, you don't have to call it.
Sets level of compression to response.
Compression level stored in config_compression setting. You can change it from admin panel.
