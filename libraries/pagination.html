Pagination Library

Pagination library used for output pagination on your page.

Pagination is not the library that you call with controller and load with loader, often it's simply called by class name.
new Pagination();
It creates several pagination links and also prev link (<), next link (>), first page link (|<), last page link (>|)

You can change properties to customize pagination and then call render() method.
Default values below are sometimes overwritten.

Properties

public int total = 0
Total amount of elements (products, articles, etc), that were found
public int page = 1
Current page, that opened
In default files current page this construction gets current page: 

    if (isset($this->request->get['page'])) {
        $page = $this->request->get['page'];
    } else {
        $page = 1;
    }
Since php7 you can simplify: $page = $this->request->get['page'] ?? 1;
public int limit = 20
Limit of elements that will be shown on the page.
public int num_links = 8
Amount of pagination links, not including next and prev arrows
public string url = ''
Base url for pagination links, including page param (with placeholder {page} - can be urlencoded)
public string text_first = '|&lt;'
Inner html of first page link (|<)
public string text_last = '&gt;|'
Inner html of last page link (>|)
public string text_next = '&gt;'
Inner html of next link (>)
public string text_prev = '&lt;'
Inner html of prev link (<)

Methods

public function render() : string
Returns the actual html of pagination, or if there only one page returns empty string ''

Example


    $pagination = new Pagination();
    $pagination->total = 67;
    $pagination->page = 2;
    $pagination->limit = 20;
    $pagination->url = $this->url->link('product/category', 'path=' . $this->request->get['path'] . '&page={page}');
