Document Library

Document library is used to take care of <head> of your page.

Basically, by using methods of this library, you pass params to header controller (or you pass scripts to footer).

It's registry-stored library, call it by $this->document

Methods

public function setTitle(string $title)
Sets document <title>
public function setDescription(string $description)
Sets document description (meta description)
public function setKeywords(string $keywords)
Sets document keywords (meta keywords)
public function addLink(string $href, string $rel)
Adds new <link> to the <head>
public function addStyle(string $href, string $rel = 'stylesheet', string $media = 'screen')
Adds new style to the <head>
public function addScript(string $href, string $postion = 'header')
Adds script to header or footer. Use $position 'footer' to pass script into footer. You can pass custom position, and then get all the scripts by method getScripts('my-position'), but what's the point?
public function setOgImage(string $image)
Sets meta open graph image

Methods below are used in header and footer controllers
public function getTitle() : string|null
Returns document title
public function getDescription() : string|null
Returns document description
public function getKeywords() : string|null
Returns document keywords
public function getLinks() : array
Return links, that were set with addLink method
public function getStyles() : array
Returns styles, that were set with addStyle method
public function getScripts(string $postion = 'header') : array
Returns scripts, added with addScript with given $position
public function getOgImage() : string|null
Return open graph image
