Get setting value

To get setting value in controller or model, use method get of Config library

It's not doing any database requests, all settings gathered on startup in core files.

To get your settings call $this->config->get(string $key)

public function get(string $key) : mixed
Get value from settings or config by key and return it. If there's no entry with such key, returns null.

Example:


    $shop_name = $this->config->get('config_name');

If you want to get setting from admin panel and have no idea with what key it's labelled, find name attribute of form element. Usually, it's the key.

If you want to get all settings in the group use method getSetting of ModelSettingSetting
