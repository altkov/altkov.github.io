Language

Files

Storage

  • Catalog

    Catalog language files stored at /catalog/language/your-language

  • Admin

    Admin language files stored at /admin/language/your-language

How file looks like

Write your entries to variable array $_.

$_['entry_enabled'] = 'Enabled';
$_['entry_name'] = 'Name';

Library

Library language can be called from controller with $this->language.

Language library methods:

public function get(string $key) : string
Gets entry from language files by.
Returns entry from language datas or, if can't find - returns $key (the one from params)
public function load(string $filename) : array
Load language file with entries by filename. First seeks file in your language directory (russian, italian, finnish). If can't find it in there, tries to load file from english directory.
Returns all language data.
If you load multiple language files in controller, language data merges (entries from last loaded files overwrites entries with same keys from early loaded files)
